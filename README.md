# Tic Tac Toe - Java Swing

A GUI-based Tic Tac Toe game built using Java Swing.  
This project demonstrates object-oriented design, event handling, and GUI development in Java.

---

## Features
- Interactive 2-player gameplay
- Clean GUI using Swing
- Win/draw detection
- Modular code structure

---

## Tech Stack
- Java
- Swing (GUI)

---

├── Resources/

│   └── GameUI/

## How to Run
1. Clone the repo:
   git clone https://github.com/Kshitij-Ranjan/tic-tac-toe-java.git

2. Open in IntelliJ / Eclipse

3. Run:
   Main.java
